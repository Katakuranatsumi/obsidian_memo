### CLAUDE.md

 - 最初にCLAUDE.mdを作成する時は、以下コマンドで生成できる。(該当アプリの情報から生成してくれる。)

Claude Codeを起動

```
claude
```

CLAUDE.mdを生成
```
/init
```
